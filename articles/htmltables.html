Table Elements and Attributes
Tag/Attribute Description
Basic Elements
<table> To enclose table content
<td> Table data - to hold table cell data
<tr> Table row - <td> elements of one row should be wrapped inside <tr>
<th> Table header - defines type of data row/column contains
Table Attributes
colspan = "span" span = No. of columns a cell spans
rowspan = "span" span = No. of rows a cell spans
scope = "col|row|colgroup|rowgroup" Used for headings(th elements) to specify to which cells 'th' are headers for - row or column or multiple rows or columns
id and headers Specify id attribute for every <th> element and headers attribute to each td elements to contain ids of all the <th> elements that acts as headings for that <td> element
Table styling elements
<colgroup> and <col> to style columns, provide style information for each column inside <col> . Blank <col> means no styling. Same styling for multiple column is specified by span.
Table Semantic elements
<caption> To add caption to the table
<thead> Wraps the heading section of the table
<tbody> Wraps the body section of the table
<tfoot> Wraps the footer section of the table. These elements are only useful when it comes to styling and doesnt add any accessibility or visual features.
